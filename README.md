# Chalk Art & Pavement Illustration Services

A professional website for Chalk Art and Pavement Illustration Services.

## Features
- **Responsive Design**: Fully compatible with desktop, tablet, and mobile devices.
- **Modern UI/UX**: Clean and artistic design with high-contrast hero sections.
- **Multiple Pages**:
  - Home (Multiple variations)
  - Services & Service Details
  - Blog & Blog Details
  - Pricing
  - About Us
  - Contact Us
  - Login & Registration
  - Dashboard
- **Branding**: Consistent branding with custom SVG favicons and color palettes.

## Tech Stack
- HTML5
- CSS3 (Vanilla CSS)
- JavaScript
- Bootstrap 5 (for components and layout)

## How to Run
Simply open `index.html` in any modern web browser.

## Project Structure
- `index.html`: Main landing page.
- `assets/`: Contains CSS, JavaScript, images, and other static files.
- `*.html`: Other service and utility pages.
